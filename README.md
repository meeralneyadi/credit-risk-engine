# Credit Risk Engine

Automated model selection and decision engine for credit risk assessment using the Home Credit Default Risk dataset.

## Project Goals
- Train and compare multiple credit risk models
- Automatically select the best model based on performance and calibration
- Convert probability of default into business decisions (Approve / Review / Reject)
- Provide explainability and monitoring
- Deploy via FastAPI

## Dataset
Home Credit Default Risk (Kaggle)

See project folders for data, notebooks, source code, and API.
